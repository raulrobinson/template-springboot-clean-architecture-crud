# Template Spring Boot Clean Architecture CRUD

- Build docker image
```shell
docker build -t my-springboot-app .
```

- Run docker container
```shell
docker run -p 8080:8080 my-springboot-app
```

- Build complete project
```shell
docker-compose up -d
```

- Stop project
```shell
docker-compose down
```
